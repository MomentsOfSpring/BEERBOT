НАРОДНЫЕ Команды:

/help - Узнать список команд
/invite - Получать ссылку-приглашение
/rules - Получить список правил
/events - Посмотреть список нерегулярных мероприятий

/plusfriends - Добавить друзей с собой
/minusfriends - Убавить друзей с собой
/pluseventfriends - Добавить друзей на НЕРЕГУЛЯРНОЕ мероприятие
/minuseventfriends - Убрать друзей с НЕРЕГУЛЯРНОЕ мероприятия


Команды для БОССОВ:
/pollnow - Ручное создание опроса
/pollres - Ручное завершение опроса
/gameon - Ручное пожелание хорошей игры
/addevent - Создать нерегулярное мероприятие
/deleteevent - Удалить нерегулярное мероприятие


Команды для АДМИНИСТРАТОРА:
/shout - Отправить анонс от имени бота (только в личном чате)
